2. Agent → Skills → enable **X Post**
3. Brain → buat task per baris di atas (tanggal + jam WIB + copy tweet)
4. Generate foto dari prompt → upload → attach ke task
5. Setelah 10 Juli, ulang dari **27 Juni** dengan variasi copy Week 2 atau buat token spotlight baru
